# projeto-front-end
O projeto servirá para atender o comércio local, através da criação de uma página web para a loja Doce História. A proposta é estimular a utilização 
de TI em pequenos comércios locais, através do desenvolvimento de websites. E para além disso, facilitar também o acesso a esses pequenos comércios. 

Através do site, o cliente consegue visualizar os produtos disponíveis, conhecer um pouco mais sobre a história da loja e com apenas um clique realizar seu cadastro.

Integrantes: Gabriela de Oliveira Landgraf, Rafaela de Oliveira Landgraf e Sarah Alves de Oliveira.
